# 本地独立刷题系统

一个功能完善的离线刷题与模拟考试系统，完全基于前端技术实现，无需服务器即可运行。所有数据存储在浏览器本地，支持多题库管理、多种学习模式、模拟考试等功能。

## ✨ 核心特性

### 📚 多模式学习
- **顺序刷题**：按题库顺序逐题练习，支持查看答案、隐藏答案、标记题目
- **随机刷题**：随机抽取题目进行练习，提高学习效率
- **背题模式**：自动显示答案和解析，适合快速记忆
- **模式切换**：支持在刷题模式和背题模式之间随时切换
- **错题复习**：自动记录错题，方便后续复习

### 🎯 模拟考试
- **灵活配置**：按题型（单选/多选/判断）独立设置题目数量和每题分值
- **计时功能**：支持自定义考试时长，实时显示剩余时间
- **考生信息**：支持输入考生姓名，考试完成后显示成绩
- **智能导航**：左侧导航栏显示所有题目，支持快速跳转和题型筛选
- **答题状态**：实时显示未作答、已作答、已标记等状态
- **交卷统计**：自动计算得分、正确率（正确题数/总题数），显示详细答题结果

### 📖 题库管理
- **多格式导入**：支持 Excel (.xlsx/.xls) 和 JSON 格式导入
- **多题库并行**：可同时管理多个题库，互不干扰
- **统计信息**：显示总题数、单选题、多选题、判断题数量
- **导出备份**：支持导出题库为 JSON 格式，方便备份和分享
- **一键操作**：从题库管理可直接进入考试模式或学习模式

### 📝 试卷生成（Word 文档）
- **随机抽题**：根据题库内容随机生成试卷，每次抽题不重复
- **灵活配置**：自定义单选题、多选题、判断题的数量和每题分数
- **方案保存**：支持将生成配置保存为方案，下次一键按方案生成
- **Word 格式**：生成  `.doc` 文件，支持双栏排版、页码、参考答案页

### 🎨 用户体验
- **键盘快捷键**：支持左右箭头键快速切换题目（←上一题，→下一题）
- **响应式设计**：适配不同屏幕尺寸，支持移动端和桌面端
- **状态可视化**：题目导航使用不同颜色标识答题状态
  - 紫色：当前题
  - 灰色：未作答
  - 蓝色：已作答
  - 黄色：已标记
  - 绿色：答对
  - 红色：答错
- **进度追踪**：实时显示答题进度条和百分比
- **离线运行**：完全支持离线环境，所有依赖可本地化

## 📖 使用指南

### 1. 导入题库

**方式一：从首页导入**
1. 点击首页的"导入题库"按钮
2. 选择 Excel 或 JSON 文件
3. 输入题库名称
4. 点击"开始导入"

**方式二：从题库管理页导入**
1. 进入"题库管理"页面
2. 点击"选择文件"
3. 选择文件并设置导入类型（考试模式/学习模式）
4. 点击"开始导入"

**题库格式要求：**
- Excel 格式：需包含列：题目、选项A、选项B、选项C、选项D、正确答案、解析、题型
- JSON 格式：需符合系统定义的数据结构
- 题型字段：`single`（单选题）、`multiple`（多选题）、`judge`（判断题）

### 2. 生成试卷

1. 在首页点击"生成试卷"按钮
2. 从下拉列表中选择题库（需先导入题库）
3. 设置各题型参数：
   - 单选题数量及每题分数
   - 多选题数量及每题分数
   - 判断题数量及每题分数
4. 点击"生成试卷"按钮，系统将自动生成 `.doc` 文件并下载到本地
5. 可选：保存当前配置为方案，下次可直接选择方案一键生成
6. 试卷打开后如显示非横向，请在 Word 中手动设置：页面布局 → 纸张大小 → A3，方向 → 横向

### 3. 模拟考试

1. 在首页点击"模拟考试"
2. 选择题库
3. 配置考试参数：
   - 单选题数量及每题分值
   - 多选题数量及每题分值
   - 判断题数量及每题分值
   - 考试时长（分钟）
4. 输入考生姓名
5. 点击"开始考试"
6. 答题过程中：
   - 使用左侧导航栏快速跳转题目
   - 点击"标记"按钮标记需要复查的题目
   - 使用左右箭头键快速切换题目
   - 查看剩余时间
7. 点击"交卷"提交考试
8. 查看考试结果：得分、正确率、详细答题情况

### 4. 顺序刷题

1. 在首页点击"顺序刷题"
2. 选择题库
3. 开始答题：
   - 选择答案后点击"下一题"
   - 点击"查看答案"查看正确答案和解析
   - 点击"隐藏答案"隐藏已显示的答案
   - 点击"标记"标记重要题目
   - 点击"背题模式"切换到背题模式（自动显示答案）
4. 查看进度：点击"查看进度"查看答题统计

### 5. 随机刷题

1. 在首页点击"随机刷题"
2. 选择题库
3. 系统随机抽取题目进行练习
4. 功能与顺序刷题相同，支持查看答案、标记、切换模式等

### 6. 题库管理

1. 进入"题库管理"页面
2. 查看所有已导入的题库列表
3. 对每个题库可以：
   - **导出**：导出题库为 JSON 文件
   - **删除**：删除不需要的题库
   - **快速进入**：点击题库卡片上的模式按钮直接进入对应模式

## 🛠️ 技术架构

| 题库管理 | `js/manage.js` | 题库列表、导出删除、界面渲染 |

```json
  ]

```

## 🐛 常见问题

### 近期修复（V1.1）

- **导入格式标签**：从首页导入 JSON 题库时，题库描述会正确显示为「导入自JSON文件」，不再误显示为 Excel。
- **方向键导航**：答题后焦点在选项上时，左右方向键仍用于切换题目，不会在选项间来回跳。
- **首屏页面显示**：首页及各子页面在 Tailwind 异步加载完成前也会正确隐藏非当前页，避免首次进入时出现多页堆叠的静态内容。

**Q: 如何备份题库？**  
A: 在题库管理页面点击"导出"按钮，将题库导出为 JSON 文件保存。

**Q: 支持哪些浏览器？**  
A: 支持所有现代浏览器，推荐使用 Chrome、Edge 或 Firefox 最新版本。

**Q: 数据存储在哪里？**  
A: 所有数据存储在浏览器的 LocalStorage 中，清除浏览器数据会丢失题库和进度。

**Q: 可以同时进行多个考试吗？**  
A: 系统支持保存考试进度，但一次只能进行一个考试。可以保存当前进度后开始新考试。

**Q: 试卷生成时提示题库题目数量不足？**
A: 生成试卷时各题型的抽取数量不能超过该题库对应题型的总题数。如果题库中单选题只有20道，就不能设置抽取30道单选题。请减少数量或导入更多题目。

**Q: 生成的试卷打开后显示纵向 A4，不是 A3 横向？**
A: 由于 HTML 伪装的 `.doc` 文件在部分 Word 版本中的兼容性问题，页面设置可能不被识别。请手动在 Word 中调整：页面布局 → 纸张大小 → A3，方向 → 横向。双栏排版即使在 A4 纵向下也能正常显示。

**Q: 页码没有显示？**
A: 页码依赖 Word 的页脚字段功能，部分版本或兼容模式下可能不生效。如未显示，可在 Word 中手动添加页脚页码。
